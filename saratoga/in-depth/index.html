about us blog style guide

in-depth

The In Depth module is a version of the horizontal card that changes the flex-grow weight from the right side to the left.

Example

After meeting Brownback’s circle, firm was on ‘right path’ to $61M no-bid contract

Thousands of emails reveal how CGI Technologies landed a $61 million contract to overhaul Kansas tax systems. New Gov. Laura Kelly has promised to crack down on no-bid contracts.

HTML

<div class="card horizontal in-depth">
  <figure>
    <div class="label sticky"><h5 class="impact">IN DEPTH</h5></div>
    <img src="https://www.kansas.com/latest-news/6aw7pv/picture225984215/alternates/LANDSCAPE_768/tax-forms.jpg">
  </figure>
  <div class="package impact">
    <h1>After meeting Brownback’s circle, firm was on ‘right path’ to $61M no-bid contract</h1>
    <summary>Thousands of emails reveal how CGI Technologies landed a $61 million contract to overhaul Kansas tax systems. New Gov. Laura Kelly has promised to crack down on no-bid contracts.</summary>
  </div>
</div>

CSS

.card.in-depth figure {
  flex-grow: 4;
}

.card.in-depth .package {
  flex-grow: 1;
}

@media (min-width: 630px) {
  .in-depth {
    min-height: 400px;
  }

	.in-depth figure {
		min-height: 300px;
	}

  .in-depth figure img {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    object-fit: cover;
    object-position: 50% 20%;
  }
}
